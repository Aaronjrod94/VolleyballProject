Cumberland Dashboards

This is a project that utilizes data from each box score of the 2021 Cumberland season. After the data was extracted, it was then cleaned up and then transformed into visualizations. These dashboards highlight the team results against their opponents and how each player performed.

    Team Record

      Player Report

      This dashboard shows every player's stats as well as a player rating that shows the impact of each player while on the court. This rating is calculated by this formula (Assists + Digs + Kills + Points + Service Aces + Blocks - Errors - Ball Handling Errors - Block Errors - Reception Errors - Service Errors)/Sets Played)
